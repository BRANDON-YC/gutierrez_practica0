(1 pts) ¿Qué diferencia existe entre HTML, CSS y JavaScript en una página web?
HTML es la estructura de la página lo que hay, CSS es el estilo cómo se ve y JavaScript es la interacción qué hace.

<html>, <head>, <title>, <meta>, <link>, <style>, <script>, <body>, <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>, <h1>…<h6>, <p>, <br>, <hr>, <a>, <img>, <ul>, <ol>, <li>, <div>, <span>, <table>, <tr>, <td>, <th>, <form>, <input> etc.

<a> → crea enlaces a otras páginas o secciones.

<img> → inserta imágenes en la web.

<form> → crea formularios para capturar datos del usuario.

Sección B: Java (10 puntos)

(1 pts) ¿Qué significa que Java es un lenguaje "orientado a objetos"?
Java organiza el codigo en clases y objetos con atributos y metodos.

(1 pts) Explica qué es la JVM (Java Virtual Machine).
Es la maquina virtual q ejecuta el bytecode en cualkier sistema.

(2 pts) Nombra los cuatro modificadores de acceso en Java y su alcance.
public (todos), protected (mismo paquete y subclases), default (mismo paquete), private (solo clase).

(1 pts) ¿Qué es una variable y cómo se declara en Java?
Es un espacio pa datos, ej: int edad = 20;.

(1 pts) ¿Cuál es la diferencia entre == y .equals() en Java?
== compara referencias, .equals() compara contenido.

(2 pts) ¿Qué es una interfaz funcional en Java? Da un ejemplo conocido.
Es una interfaz con un solo metodo abstracto, ej: Runnable.

(2 pts) ¿Para qué sirven las expresiones lambda en Java?
Sirven pa funciones cortas y limpias sin clases anonimas.

(2 pts) ¿Qué ventaja ofrecen los genéricos en Java al momento de programar?
Dan reuso de codigo y seguridad de tipos, ej: List<String>.
